Project Idea

Overview

The project idea which I would like to create is a smartphone app that allows for security measures in the house. This app would allow the user to be able to tab into security cameras. The security cameras will be able to be accessed by the user showing live footage and also provides and alarm system which would allow the user to contact the closest police department for any suspicious activities. Furthermore, the user would be able to open and close doors and windows around the household as well as being able to lock and unlock them.

Motivation

This project idea would be useful because due to advances technologies it would allow for a lower rate of crime rates around the world because through this app it would allow for the police department to be contacted so that they are on site to a robbery as soon as possible. Furthermore, through a manual locking system through a touch of a button because the user might have forgotten to lock the door this would prevent intruders from simply walking inside of their household and by the time that they are able to get inside the house the cameras would have detected movement and alerted the user through the app.

Description

The project idea is to design a security system for around the household which would consist of cameras and locks. The cameras would be placed around the household to the users liking and would be able to provide live footage to the app on what is currently happening during the moment. If the user is not on the app, they would be alerted if there are any suspicious activities happening in the area. Furthermore, the lock system would be able to detect and alert the user if it has been tampered with and would automatically lock itself if the user has left the house and had forgotten to lock it. This would also alert the user to remind them that the door has been automatically locked. This system would operate through a plug which connects to any power plug around the house. There will also be a pack up battery which would temporarily be used if there is a blackout or power outage around the household. The camera would be able to record footage and keep it safely stored for 24 hours which it will then delete and update with the new camera footage. The project would be sold in a set with 4 cameras and 4 locks with simple instructions on how to install and connect that app to the cameras within the household.

Tools and Technologies

Hardware that would be used be the physical camera which would be programmed by software allowing users to watch live footage of what is currently happening through their app on their smartphone. Other equipment required would be a backup battery which would allow the camera to be used during a blackout or power outage as it would be an additional and temporary source of power for the system to operate.

Skills Required

A software engineer will be required to help design and implement the necessary software allowing the cameras to operate and provide live footage to the user via the app. Furthermore, someone experienced with GitHub will be required as they will be able to save and store footage provided by the camera as well as being able to use and allow users to access camera footage as well as being able to lock doors automatically if the user as left the premises.

Outcome

If the project idea is successful it would allow for users to feel safer in and out of their household knowing that they will be able to have extra security measures around the household. This would also likely decrease the rates of house robberies because the app would be able to alert the police department if there is any suspicious activity and so the police would be able to be on site as soon as possible.
